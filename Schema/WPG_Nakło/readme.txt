Program należy uruchomić z paramterem:
	
	-w 					=> oznacza, że ma wykonać walidację. Bez tego parametru po prostu wczyta dane do XLSX
	
	-s C:\folderXML		=> katalog w którym są pliki XML. Program czyta dane ze wskazanego katalogu jak i podfolderów w nim zanjdujących się 

ReadOpXML.exe -w -s c:\XML
